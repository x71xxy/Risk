## 技术栈
- 后端：Flask 3.0.0
- 数据库：MySQL + SQLAlchemy
- 前端：原生HTML/CSS/JavaScript
- 用户认证：Flask-Login
- 表单处理：Flask-WTF
- 邮件服务：Flask-Mail
- 数据库迁移：Flask-Migrate

## 注意事项
1. .env文件包含敏感信息，部署时需要妥善保管
2. 数据库迁移文件需要重新初始化
3. 邮件服务使用QQ邮箱，建议更换为企业邮箱
4. 需要完成 AWS VPC 配置
5. 需要准备演示视频
